URL Shortener

Give a web-address as a parameter and recieve a new shortened URL in return.
From then on that shortened URL will redirect you to the web-address you specified as a parameter.

E.g. typing:

https://fcc-urlshortener.glitch.me/www.google.com

could return you the following:

{"Original Url":"www.google.com","Shortened Url":"https://fcc-urlshortener.glitch.me/2031"}

Now typing https://fcc-urlshortener.glitch.me/2031 would redirect you to Google.

Please note, each user (i.e. IP address), has their own unique set of shortened URLs for each website they give to this microservive.
